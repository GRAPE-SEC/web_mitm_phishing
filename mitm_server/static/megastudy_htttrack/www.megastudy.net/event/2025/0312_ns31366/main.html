  • 본 이벤트는 고1~N수생 회원 대상 (생년월일이 2001년~2009년생)으로 진행됩니다.

  • 메가캐쉬, 무료 배송 쿠폰은 ID 당 1회 지급 및 [마이페이지>캐쉬/할인권]에서 확인 가능하며, 사용 기간은 10일입니다. (기간 연장 및 재지급 불가)
  • 휴대전화번호 본인 인증은 ID 당 1회만 가능합니다. (번호 중복 인증 및 참여 불가)
  • ID 당 1개의 반만 개설 또는 참여할 수 있으며, 반을 개설한 후에는 변경 또는 삭제할 수 없으니 신중히 개설해 주세요.

  • 3월 학평 당일 18시부터 선착순 1,000반 신청이 시작되며, 당일 20시부터 보너스 300반 신청이 시작됩니다.
